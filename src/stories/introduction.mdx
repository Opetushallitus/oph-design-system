# OPH Design System

OPH Design System (ODS) is a design system used in web applications for the [Finnish National Agency for Education (OPH)](https://www.oph.fi).

## Reusable components

ODS provides a set of reusable [React](https://react.dev) components that can be used to build web applications with consistent styles and user experience.

## Consistent styles

ODS components are designed to follow ODS guidelines. This helps developers to build applications that have consistent look and feel.

## Based on Material-UI

ODS implementation is based on [Material-UI](https://mui.com/material-ui/), a popular React component library.
